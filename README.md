# 📊 Student Performance Analysis Using Python

## 📌 Project Overview

This project focuses on analyzing the **Student Performance Dataset** using Python. The objective is to explore, clean, analyze, and visualize the dataset to gain meaningful insights into students' academic performance.

The project follows a complete data analysis workflow, including data loading, preprocessing, exploratory data analysis (EDA), visualization, and interpretation of results.

---

## 🎯 Objectives

- Load the Student Performance dataset
- Explore and clean the data
- Identify missing values and duplicate records
- Analyze students' final grades (G3)
- Examine the relationship between study time and academic performance
- Compare average grades based on gender
- Create informative visualizations
- Draw meaningful conclusions from the analysis

---

## 🛠️ Technologies Used

- Python
- Google Colab / Jupyter Notebook
- Pandas
- NumPy
- Matplotlib

---

## 📂 Dataset

Dataset Used:
- **student-mat.csv**

The dataset contains information about students, including demographic details, family background, study habits, and academic performance.

---

## 📊 Analysis Performed

- Dataset exploration
- Data cleaning
- Missing value detection
- Duplicate removal
- Average final grade calculation
- Students scoring above 15
- Correlation between study time and final grades
- Gender-wise average performance

---

## 📈 Visualizations

The project includes the following visualizations:

- Histogram of Final Grades
- Scatter Plot (Study Time vs Final Grade)
- Bar Chart (Average Grade by Gender)

---

## 📋 Key Findings

- Dataset contains **395 student records** and **33 features**.
- No missing values or duplicate records were found.
- Average final grade (G3): **10.42**
- **40 students** scored above **15**.
- Study time has a **very weak positive correlation** with final grades.
- Male students performed slightly better on average than female students.
